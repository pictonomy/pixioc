addProcessingMethod()method 
public function addProcessingMethod(processingMethod:Function, ... args):void

Adds new preprocessing method.

Preprocessing act after xml loading and before context parsing. Used to transform context data on fly.

Parameters

processingMethod:Function — Method to call. Method must accept an XML object as first argument and return an XML object when completed.
 
... args — Additionals parameters to pass to method when executing.

See also

net.pixlib.ioc.context.processor.ProcessingHelper

Example 
		 
		 var oLoader : ApplicationLoader = new ApplicationLoader( this, false );
		 oLoader.addProcessingMethod( XMLProcessingHelper.changeObjectAttribute, "cross", "visible", "false" );
		 oLoader.addProcessingMethod( XMLProcessingHelper.changePropertyValue, "cross", "x", 600 );
		 oLoader.addProcessingMethod( XMLProcessingHelper.addResource, "newStyle", "myStyle.css" );
addProcessingVariable()method 
public function addProcessingVariable(name:String, value:*, flashvarName:String = null):void

Sets new pair name / value for variable replacement when context file will be laoded.

Parameters

name:String — Name of the variable to replace.
 
value:* — New value for this variable.
 
flashvarName:String (default = null)


Example
Variable definition somewhere in the xml context : 
		 
		 ${POSITION_X}

Example
Setting this variable using setVariable() method : 
		 
		 rtLoader.addProcessingVariable( "POSITION_X", 200 );
